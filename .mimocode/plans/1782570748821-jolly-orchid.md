# Raw-Records Modal on Hos-List Page

## Summary
Click a non-zero month cell on `/dashboard/hos-list` → modal opens showing raw records filtered by `hospcode` + month + fiscal year, with server-side pagination (50 per page). All columns shown except `cid_aes` and `log_import_id`.

## Step 1 — New API Endpoint

**File**: `webapp/app/api/raw-records/route.js` (create)

**GET params**: `file`, `hospcode`, `monthValue` (e.g. `"10"`), `fiscalYear` (Buddhist, e.g. `"2569"`), `page`, `limit`

**Logic**:
1. Auth via `requireApiJwt`
2. Validate `file` exists in `c_file`; validate `hospcode` and `monthValue` provided
3. Detect date column via `chooseMonthlyDateColumn(columns)`
4. Compute `fiscalYearAd = normalizeFiscalYear(fiscalYear)`, then `getFiscalYearRange(fiscalYearAd)`
5. Query `information_schema.COLUMNS` for the table, filter out `cid_aes` and `log_import_id` in JS
6. COUNT query with WHERE: `hospcode = ?` + `SUBSTRING(LEFT(dateCol,8), 5, 2) = ?` + fiscal year date range
7. SELECT query with same WHERE + `LIMIT ? OFFSET ?`, selecting only allowed columns
8. Return `{ total, page, limit, columns, rows }`

**Imports from `dashboard-data.mjs`**:
`chooseMonthlyDateColumn`, `datePrefixExpression`, `getFiscalYearRange`, `normalizeFiscalYear`, `quoteIdentifier`

**Response shape**:
```json
{
  "total": 9285,
  "page": 1,
  "limit": 50,
  "columns": ["hospcode", "pid", "hn", "seq", "date_serv", "time_serv", ...],
  "rows": [{ "hospcode": "11251", "pid": "...", ... }, ...]
}
```
